Windows 高阶指南导言

——与其感慨路难行，不如马上出发

想说点什么

入职两个月，我一直在探寻内心深处的热爱与追求。刚入职时，诸多不适应让我满心只想逃离，如今虽已逐渐适应了这忙碌的生活节奏，可除了每月到账的工资，我究竟还收获了什么？

回首过往学习岁月，自初二初次接触计算机起，我便在知识的海洋里不断探索。初中时，安卓手机刷机风潮正盛，我怀揣着亲手编译出手机系统的梦想，沉浸其中。然而，当我终于具备了这番能力，却发现“刷机”已成往事。此后，我涉足诸多领域，Linux 操作系统开发、k8s、前后端开发、企业组网等技术知识，数据结构、抽象代数、粗糙模糊集等数学理论，皆有所涉猎，可始终未能觅得那份能让我倾尽一生去坚守的挚爱，想来不免有些惭愧。

不过，这些年的学习与折腾并非徒劳，我积累了诸多宝贵经验。与其任由它们在脑海中蒙尘，不如将它们整理成册，分享给更多人。与其空叹前行之路艰难，不如即刻启程。哪怕这条道路荆棘丛生，也定会让我收获满满。

经过一番思量，我决定先从 Windows 使用经验入手，毕竟其受众广泛，或许能吸引众多目光。但这只是一个起点，绝非终点。

我会带给你这些

网络上已经有非常多的 Windows 教程，出版物更是数不过来，微软官方也有自己的文档。但是，大多数的教程都过于着重理论教学，忽略了实践。微软官方文档必然是最详细最权威，但是阅读门槛过高，非专业人士很难读懂，并且文档涉及到第三方程序的问题。

我会将我在使用 Windows（包括 Windows Server）过程中遇到的问题进行记录与解决。从安装系统开始，到舒适地使用系统结束。

到最后，我会带你搭建一个完善的适用于多用户的 Windows 专业版系统，以及一个可以用作 NAS 的服务器系统。
